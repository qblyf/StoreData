# 动作分析指标更新文档

## 更新概述

1. 在动作分析的"巡检评分"指标中新增了"卫生评分"子指标
2. 新增与"巡检评分"平级的"营销评分"指标，包含企业微信、抖音曝光、1公里营销三个子指标
3. 新增与"巡检评分"平级的"即时零售"指标，包含美团订单、京东销售两个子指标

## 更新内容

### 1. 巡检评分子指标扩展

巡检评分现在包含三个子指标：

1. **客诉处理** - 统计周期内客户投诉总数
2. **展陈质量** - 陈列规范度、视觉效果、更新频率
3. **卫生评分** - 门店清洁度、产品整洁度、环境维护（新增）

### 2. 营销评分（新增）

营销评分是与巡检评分平级的综合评分指标，包含三个子指标：

1. **企业微信** - 企业微信好友总数
2. **抖音曝光** - 视频总播放量
3. **1公里营销** - 周边社区营销活动触达人数

### 3. 即时零售（新增）

即时零售是与巡检评分、营销评分平级的综合评分指标，包含两个子指标：

1. **美团订单** - 美团平台订单数
2. **京东销售** - 京东平台销售额

- **指标ID**: `hygiene_score`
- **指标名称**: 卫生评分
- **单位**: 分
- **计算公式**: 卫生评分 = 门店清洁度 + 产品整洁度 + 环境维护
- **目标标准**: 90分以上
- **检查频率**: 每日检查记录
- **数据范围**: 85-98分（模拟数据）

### 3. 卫生评分详情

- **指标ID**: `hygiene_score`
- **指标名称**: 卫生评分
- **单位**: 分
- **计算公式**: 卫生评分 = 门店清洁度 + 产品整洁度 + 环境维护
- **目标标准**: 90分以上
- **检查频率**: 每日检查记录
- **数据范围**: 85-98分（模拟数据）

### 4. 营销评分详情

- **指标ID**: `marketing_score`
- **指标名称**: 营销评分
- **单位**: 分
- **计算公式**: 营销评分 = 各项营销指标加权平均分
- **目标标准**: 85分以上
- **评估维度**: 企业微信、抖音曝光、1公里营销等多维度评估
- **数据范围**: 80-95分（模拟数据）

#### 4.1 企业微信子指标

- **指标ID**: `wechat_followers`
- **指标名称**: 企业微信
- **单位**: 人
- **计算公式**: 企业微信好友总数
- **目标标准**: 月增长目标50人，活跃度保持60%以上
- **数据范围**: 200-800人（模拟数据）

#### 4.2 抖音曝光子指标

- **指标ID**: `douyin_views`
- **指标名称**: 抖音曝光
- **单位**: 次
- **计算公式**: 统计周期内视频总播放量
- **目标标准**: 月目标10万次播放，提升品牌曝光度
- **数据范围**: 50,000-200,000次（模拟数据）

#### 4.3 1公里营销子指标

- **指标ID**: `local_marketing`
- **指标名称**: 1公里营销
- **单位**: 人
- **计算公式**: 周边社区营销活动触达人数
- **目标标准**: 月目标触达500人，提升周边知名度
- **数据范围**: 300-800人（模拟数据）

### 5. 即时零售详情

- **指标ID**: `instant_retail`
- **指标名称**: 即时零售
- **单位**: 分
- **计算公式**: 即时零售 = 各平台订单量和销售额加权平均分
- **目标标准**: 85分以上
- **评估维度**: 美团订单、京东销售等平台业务评估
- **数据范围**: 80-95分（模拟数据）

#### 5.1 美团订单子指标

- **指标ID**: `meituan_orders`
- **指标名称**: 美团订单
- **单位**: 单
- **计算公式**: 统计周期内美团订单总数
- **目标标准**: 月增长目标10%，优化评分和服务
- **数据范围**: 50-200单（模拟数据）

#### 5.2 京东销售子指标

- **指标ID**: `jd_sales`
- **指标名称**: 京东销售
- **单位**: 元
- **计算公式**: 统计周期内京东销售总额
- **目标标准**: 月目标根据品类设定，保持稳定增长
- **数据范围**: 10,000-50,000元（模拟数据）

### 6. 卫生评分评估维度

卫生评分包含以下三个维度：

1. **门店清洁度**
   - 地面清洁
   - 墙面整洁
   - 天花板无污渍
   - 门窗明亮

2. **产品整洁度**
   - 展示产品无灰尘
   - 产品摆放整齐
   - 包装完好
   - 标签清晰

3. **环境维护**
   - 空气清新
   - 照明充足
   - 温度适宜
   - 无异味

### 7. 营销评分评估维度

营销评分包含以下三个维度：

1. **企业微信运营**
   - 好友数量增长
   - 用户活跃度
   - 互动频率
   - 内容质量

2. **抖音内容营销**
   - 视频播放量
   - 点赞互动数
   - 粉丝增长
   - 内容创意

3. **本地化营销**
   - 周边社区覆盖
   - 活动触达人数
   - 转化效果
   - 品牌认知度

### 8. 即时零售评估维度

即时零售包含以下两个维度：

1. **美团平台运营**
   - 订单数量增长
   - 订单转化率
   - 用户评分
   - 配送时效
   - 客户复购率

2. **京东平台运营**
   - 销售额增长
   - 商品评价
   - 物流满意度
   - 活动参与度
   - 品类拓展

## 使用方法

### 查看卫生评分

1. 打开Dashboard
2. 点击顶部的"动作分析"标签页
3. 找到"巡检评分"指标卡片
4. 点击卡片查看详情，可以看到三个子指标，包括新增的"卫生评分"

### 查看营销评分

1. 打开Dashboard
2. 点击顶部的"动作分析"标签页
3. 找到"营销评分"指标卡片（与巡检评分平级）
4. 点击卡片查看详情，可以看到三个子指标：
   - 企业微信
   - 抖音曝光
   - 1公里营销

### 查看即时零售

1. 打开Dashboard
2. 点击顶部的"动作分析"标签页
3. 找到"即时零售"指标卡片（与巡检评分、营销评分平级）
4. 点击卡片查看详情，可以看到两个子指标：
   - 美团订单
   - 京东销售

### 数据展示

- 卫生评分会显示在巡检评分的子指标列表中
- 营销评分作为独立指标显示，与巡检评分、案例贡献等指标并列
- 即时零售作为独立指标显示，与巡检评分、营销评分等指标并列
- 所有指标都支持查看历史趋势
- 支持门店对比
- 支持目标达成率分析
