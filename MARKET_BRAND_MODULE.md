# 市场品牌分析模块

## 功能概述

市场品牌分析模块重点展示门店销量与区域市场销量的对比分析，帮助管理者了解门店在区域市场中的竞争地位和各品牌的表现。

## 核心功能

### 1. 市场销量对比

展示门店与区域市场的整体对比数据：

- **门店销量**: 显示门店总销量和在区域市场的占比
- **区域市场销量**: 显示区域市场总销量和竞争门店数量
- **区域排名**: 门店在区域内的排名位置
- **市场占有率**: 门店在区域市场的份额百分比

支持按时间周期筛选：
- 本月
- 本季度
- 本年

### 2. 品牌销量对比分析

详细的品牌级别对比表格，包含：

| 列名 | 说明 |
|------|------|
| 品牌 | 手机品牌名称和图标 |
| 门店销量 | 该品牌在门店的销量 |
| 门店占比 | 该品牌在门店总销量中的占比 |
| 区域市场销量 | 该品牌在区域市场的总销量 |
| 市场占比 | 该品牌在区域市场的占比 |
| 对比分析 | 门店占比与市场占比的差异 |

**对比分析指标**：
- 🟢 优于市场：门店该品牌占比高于市场平均水平
- 🔴 低于市场：门店该品牌占比低于市场平均水平

### 3. 可视化图表

- **门店各品牌销量图**: 展示门店内各品牌的销量分布
- **区域市场各品牌销量图**: 展示区域市场各品牌的销量分布
- 两图并列对比，直观展示差异

### 4. 市场洞察

基于数据分析提供三类洞察：

- **💡 优势品牌**: 识别门店表现优于市场的品牌，建议加大投入
- **⚠️ 待提升品牌**: 识别门店表现低于市场的品牌，建议优化策略
- **🎯 市场机会**: 基于市场占有率和排名，提供提升建议

## 数据示例

### 市场对比数据
```typescript
{
  storeTotalSales: 3600,      // 门店总销量（台）
  marketTotalSales: 45000,    // 区域市场总销量（台）
  storeMarketShare: 8.0,      // 门店市场占有率（%）
  storeRanking: 3,            // 门店排名
  totalCompetitors: 25,       // 竞争门店数
}
```

### 品牌数据
```typescript
{
  brand: '苹果',
  storeSales: 1250,           // 门店销量
  marketSales: 15000,         // 市场销量
  storeShare: 34.7,           // 门店占比（%）
  marketShare: 33.3,          // 市场占比（%）
  icon: '🍎'
}
```

## 使用场景

1. **竞争分析**: 了解门店在区域市场的竞争地位
2. **品牌策略**: 识别优势品牌和待提升品牌，优化库存和促销
3. **市场定位**: 根据市场占有率和排名，制定提升策略
4. **业绩评估**: 对比门店与市场表现，评估经营效果

## 设计特点

- **对比视角**: 始终以门店 vs 市场的对比视角展示数据
- **可视化**: 使用进度条、颜色编码等方式直观展示对比结果
- **洞察驱动**: 不仅展示数据，还提供基于数据的业务洞察
- **响应式设计**: 适配不同屏幕尺寸，移动端友好
