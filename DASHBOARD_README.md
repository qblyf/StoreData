# 门店数据指标看板

## 概述

这是一个基于 React + TypeScript 的门店数据指标看板系统，用于实时展示和监控门店的财务指标和业务指标。

## 已实现功能

### ✅ 任务 4.1: KPICard 组件
- 显示指标名称、当前值、单位
- 显示目标值和完成率（进度条）
- 显示环比/同比变化（带上升/下降箭头）
- 未达标状态高亮显示（红色边框和背景）
- 支持 null 值显示为 "N/A"

### ✅ 任务 4.2: KPI 卡片网格布局
- 分为"财务指标"和"业务指标"两个区域
- 财务指标：毛利、毛利率、利润、净利率
- 业务指标：成交率、进店率、人工产出、毛利工费比
- 响应式网格布局，自动适配不同屏幕尺寸

### ✅ 任务 11.1: Dashboard 主组件
- 整合 KPIGrid 组件
- 门店选择器（下拉菜单）
- 门店信息展示（类型、级别、员工数）
- 数据加载状态（loading spinner）
- 页面标题和副标题
- 页脚显示数据更新时间

### ✅ 任务 11.3: 样式和主题
- 统一的紫色渐变配色方案
- 卡片式设计，带阴影和悬停效果
- 完整的响应式设计（支持桌面、平板、手机）
- 移动端优化布局
- 流畅的动画和过渡效果

## 技术栈

- **React 18**: UI 框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具
- **Vitest**: 测试框架
- **CSS3**: 样式（包含 Grid、Flexbox、动画）

## 核心功能说明

### 1. 指标展示
- **财务指标**: 展示门店的盈利能力（毛利、利润、利润率等）
- **业务指标**: 展示门店的运营效率（成交率、人工产出等）

### 2. 目标监控
- 每个指标可设置目标值
- 实时显示目标完成率（百分比和进度条）
- 未达标指标自动高亮提醒

### 3. 趋势分析
- 显示环比变化百分比
- 用颜色和箭头直观表示增长/下降趋势

### 4. 多门店支持
- 支持切换查看不同门店数据
- 显示门店基本信息（类型、级别、员工数）
- 8 个预设门店数据

### 5. 响应式设计
- **桌面端**: 4 列网格布局
- **平板端**: 2-3 列网格布局
- **手机端**: 单列布局，优化触摸操作

## 数据说明

当前使用模拟数据，包含：
- 8 个门店（旗舰店、标准店、迷你店）
- 20+ 个指标（财务、业务、客流等）
- 自动生成的时间序列数据
- 基于门店类型和级别的差异化数据
