# 门店数据指标看板 - 视觉设计说明

## 整体设计风格

### 配色方案
- **主色调**: 紫色渐变 (#667eea → #764ba2)
- **背景**: 渐变背景，营造现代科技感
- **卡片**: 白色背景 (#ffffff)，带阴影和圆角
- **文字**: 深灰色 (#1a1a1a) 主文字，浅灰色 (#666, #999) 辅助文字

### 状态颜色
- **成功/达标**: 绿色 (#52c41a)
- **警告/未达标**: 红色 (#ff4d4f)
- **增长**: 绿色背景 (#f6ffed)
- **下降**: 红色背景 (#fff1f0)
- **进度中**: 橙色渐变 (#ffa940 → #fa8c16)

## 交互效果

### 悬停效果
- 卡片阴影加深
- 卡片轻微上浮（translateY: -2px）
- 过渡动画 0.3s

### 进度条动画
- 宽度变化带动画效果
- 达标时变为绿色渐变
- 未达标时保持橙色渐变

### 下拉菜单
- 聚焦时显示紫色边框
- 带阴影效果
- 选项悬停高亮

## 响应式断点

### 桌面端 (> 1200px)
- 4 列网格布局
- 卡片最小宽度 280px
- 间距 20px

### 平板端 (768px - 1200px)
- 2-3 列自适应网格
- 卡片最小宽度 250px
- 间距 16px

### 手机端 (< 768px)
- 单列布局
- 卡片全宽
- 间距 12px
- 工具栏垂直排列

## 字体规范

### 标题
- 主标题: 28px, 粗体 (700)
- 副标题: 14px, 常规 (400)
- 区域标题: 18px, 半粗 (600)

### 卡片内容
- 指标名称: 14px, 半粗 (500)
- 数值: 32px, 粗体 (700)
- 单位: 16px, 常规 (400)
- 辅助信息: 13px, 常规 (400)

## 图标和符号

- ↑ 上升箭头（绿色）
- ↓ 下降箭头（红色）
- [未达标] 警告标签（红色背景）
- ▼ 下拉箭头

## 动画时长

- 悬停效果: 0.3s
- 进度条: 0.3s
- 页面过渡: 0.3s
- 加载动画: 1s (循环)

## 阴影规范

- 卡片默认: 0 2px 8px rgba(0, 0, 0, 0.08)
- 卡片悬停: 0 4px 16px rgba(0, 0, 0, 0.12)
- 头部: 0 2px 12px rgba(0, 0, 0, 0.08)

## 圆角规范

- 卡片: 12px
- 按钮/输入框: 8px
- 进度条: 4px
- 标签: 4px
