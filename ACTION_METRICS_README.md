1. **卫生评分** 🆕

   - 类型: 综合评分
   - 子指标: 美团订单、京东销售
   - 目标: 85分以上

## 🎨 如何使用

### 在Dashboard中查看

1. **打开应用**
   ```bash
   npm run dev
   ```

2. **导航到动作分析**
   - 点击顶部导航栏的"动作分析"标签页

3. **查看综合评分**
   - 找到带有⭐标记的指标卡片
   - 巡检评分、营销评分、即时零售

4. **查看子指标**
   - 点击综合评分卡片
   - 展开查看各个子指标的详细数据

### 指标卡片功能

- **点击卡片** - 查看详细信息
- **查看趋势** - 历史数据趋势图
- **门店对比** - 不同门店的对比分析
- **目标达成** - 与目标值的对比

## 🔧 开发指南

### 添加新的子指标

1. 在 `KPIGrid.tsx` 中找到对应的综合评分指标
2. 在 `subMetrics` 数组中添加新的子指标配置
3. 在 `mockDataGenerator.ts` 中添加数据生成逻辑

### 添加新的综合评分指标

1. 在 `KPIGrid.tsx` 的 `actionMetrics` 数组中添加新指标
2. 设置 `subMetrics` 属性定义子指标
3. 在 `mockDataGenerator.ts` 中添加数据生成逻辑

## ❓ 常见问题

### Q1: 为什么要引入综合评分？
**A**: 综合评分可以提供整体视图，便于快速评估某个领域的整体表现，同时通过子指标提供细节分析。

### Q2: 子指标可以独立查看吗？
**A**: 可以。点击综合评分卡片后，可以查看各个子指标的详细数据和趋势。

### Q3: 如何调整指标权重？
**A**: 目前使用默认权重。后续版本将支持自定义权重配置。

### Q4: 数据多久更新一次？
**A**: 当前使用模拟数据，实时生成。集成真实数据后，更新频率取决于数据源。
