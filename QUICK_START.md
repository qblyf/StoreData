# 快速启动指南

## 查看业务分析模块

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 访问Dashboard

打开浏览器访问 `http://localhost:5173`

### 3. 导航到业务分析

1. 在Dashboard顶部，你会看到三个标签：
   - 💰 财务分析
   - 📊 业务分析
   - ⚡ 动作分析

2. 点击 **"📊 业务分析"** 标签

### 4. 浏览业务模块

点击业务分析后，你会看到10个业务模块的导航标签：

- 📈 整体销售分析
- 🏷️ 市场销量与品牌
- 💼 主营分析
- 📡 运营商分析
- 👥 付费会员分析
- ♻️ 回收分析
- 📱 二手机分析
- ⌚ 智能产品分析
- 🔌 配件分析
- 🔧 维修分析

### 5. 查看模块详情

点击任意模块标签，查看该模块的：
- 核心KPI指标卡片
- 趋势图表
- 对比分析图表

## 功能特性

### 门店切换
- 使用顶部的门店选择器切换不同门店
- 所有模块数据会根据选择的门店更新（未来功能）

### 数据导出
- 点击"导出数据"按钮导出CSV格式数据
- 点击"导出图表"按钮导出PNG格式图片

### 响应式设计
- 支持桌面端和移动端访问
- 自动适配不同屏幕尺寸
