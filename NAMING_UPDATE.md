# 指标命名更新说明

## 📝 更新内容

将"营销评分"指标名称更改为"运营评分"，以更准确地反映该指标的评估内容。

---

## 🔄 变更详情

### 指标信息

- **指标ID**: `marketing_score` (保持不变)
- **原名称**: 营销评分
- **新名称**: 运营评分 ✅
- **描述**: 门店运营综合得分
- **公式**: 运营评分 = 各项运营指标加权平均分

### 变更原因

该指标包含的子指标（企业微信、抖音曝光、1公里营销）更侧重于日常运营活动，而非纯粹的营销活动，因此将名称调整为"运营评分"更为准确。

---

## 📊 指标体系

### 动作分析四大综合评分

```
动作分析
│
├─ 🏪 运营分析
│  ├─ ⭐ 巡检评分 (3个子指标)
│  └─ 用户评分
│
├─ 📱 营销分析
│  ├─ ⭐ 运营评分 (3个子指标) ← 名称更新
│  └─ ⭐ 即时零售 (2个子指标)
│
└─ 👥 员工成长分析
   └─ ⭐ 团队建设 (5个子指标)
```

### 运营评分的子指标

1. **企业微信** (wechat_followers)
   - 企业微信好友总数
   - 月增长目标50人

2. **抖音曝光** (douyin_views)
   - 视频总播放量
   - 月目标10万次播放

3. **1公里营销** (local_marketing)
   - 周边社区营销活动触达人数
   - 月目标触达500人
