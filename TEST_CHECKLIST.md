# 业务分析整合测试清单

### 1. 启动应用
```bash
npm run dev
```
- [ ] 服务器成功启动
- [ ] 没有编译错误
- [ ] 浏览器自动打开或访问 http://localhost:5173

### 2. 检查主Dashboard
- [ ] 看到"门店数据指标看板"标题
- [ ] 看到门店选择器
- [ ] 看到三个主标签：💰 财务分析、📊 业务分析、⚡ 动作分析

### 3. 测试财务分析（确保未受影响）
- [ ] 点击"💰 财务分析"标签

- [ ] 看到子导航栏（白色背景，圆角）
- [ ] 看到11个子标签：
  - [ ] 📊 业务概览
  - [ ] 📈 整体销售
  - [ ] 🏷️ 市场品牌

  - [ ] 🔧 维修

### 5. 测试业务概览（默认）
- [ ] 默认显示"业务概览"（蓝紫色渐变背景）
- [ ] 看到"业务数据分析"标题
- [ ] 看到5个KPI卡片：
  - [ ] 过店客流
  - [ ] 进店客流
  - [ ] 进店率
  - [ ] 成交客户数
  - [ ] 成交率
- [ ] 每个卡片显示数值、单位、目标、完成度

### 6. 测试整体销售模块
- [ ] 点击"📈 整体销售"子标签

- [ ] 看到"核心指标"部分

  - [ ] 销售订单数：3,580单
  - [ ] 客单价：349元
  - [ ] 同比增长：15.2%
- [ ] 看到"销售趋势"图表区域
- [ ] 看到"销售对比"图表区域

### 7. 测试市场品牌模块
- [ ] 点击"🏷️ 市场品牌"子标签
- [ ] 内容切换到市场品牌模块
- [ ] 看到品牌销量KPI卡片：

  - [ ] OPPO销量
- [ ] 看到品牌市场份额图表区域

- [ ] 💼 主营分析 - 显示主营收入、利润等指标
- [ ] 📡 运营商 - 显示三大运营商业务量
- [ ] 👥 会员 - 显示会员相关指标
- [ ] ♻️ 回收 - 显示回收业务指标

- [ ] 点击"⚡ 动作分析"标签
- [ ] 看到员工业绩表格

- [ ] 功能正常，未受影响

#### 桌面端（>768px）
- [ ] 子导航标签横向排列

#### 移动端（<768px）
- [ ] 缩小浏览器窗口到手机尺寸
- [ ] 子导航标签可以横向滚动
- [ ] 标签大小适中，易于点击
- [ ] 内容区域自适应宽度

### 11. 测试交互效果
- [ ] 鼠标悬停子标签时有hover效果（背景变浅、轻微上移）
- [ ] 点击子标签时有点击反馈
- [ ] 激活的子标签有明显的视觉区分（渐变背景）
- [ ] 切换子标签时内容平滑切换

### 12. 测试浏览器兼容性
- [ ] Chrome - 功能正常
- [ ] Firefox - 功能正常
- [ ] Safari - 功能正常
- [ ] Edge - 功能正常

### 13. 测试性能
- [ ] 切换子标签响应迅速（<300ms）
- [ ] 没有明显的卡顿
- [ ] 内存使用正常
- [ ] CPU使用正常

### 14. 测试数据显示
- [ ] 业务概览的数据来自Dashboard
- [ ] 业务模块的数据正确显示
- [ ] KPI卡片格式正确（数值、单位、趋势）
- [ ] 目标完成度正确显示

### 15. 测试边界情况
- [ ] 快速连续点击子标签 - 不出错
- [ ] 在不同主标签间切换 - 状态保持正确
- [ ] 刷新页面 - 默认显示业务概览
